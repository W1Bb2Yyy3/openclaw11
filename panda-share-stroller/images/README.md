### 图片格式
- **网络传播**：JPG格式，压缩率80-85%

### 文件大小控制
- **封面图**：200KB以内
- **内容图**：100KB以内

### 分辨率规范

## 🛠️ 制作建议

### 产品拍摄要点
1. **主体突出**：确保童车在画面中央位置
2. **清晰展示**：产品细节清晰可见
3. **角度多样化**：拍摄不同角度，展示全貌
4. **场景化拍摄**：在实际使用环境中拍摄

### 图片优化技巧
1. **色彩调整**：确保品牌色调一致性
2. **亮度平衡**：避免过曝或过暗
3. **裁剪优化**：确保主体构图合理
4. **添加水印**：保护品牌知识产权

### 发布前检查
- ✅ 文件大小是否合适

- ✅ 品牌标识是否正确
