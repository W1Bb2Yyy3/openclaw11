```bash

# 负面提示

## 🎯 优化提示词技巧

### 1. 结构化提示词
```
[主体描述] + [场景环境] + [视觉风格] + [技术参数] + [质量要求] + [排除要素]
```

### 2. 具体案例
```

# 场景使用图

### 3. 参数控制
```
# 尺寸控制
--ar 16:9  # 16:9宽屏
--ar 9:16  # 9:16竖屏  
--ar 1:1   # 1:1正方形

# 质量控制

--style raw # 原始风格
--chaos 10 # 创意性
```

## 🔧 故障排除指南

### 常见问题解决

#### 1. 图片质量不佳
**原因提示词不够详细**
**解决方案**：
```
优化提示词：
- 添加质量词汇：ultra high definition, detailed, professional
- 指定摄影风格：product photography, studio lighting
- 排除负面元素：--no blurry, low quality
```

#### 2. 产品变形失真
**原因生成参数设置不当**
**解决方案**：
```
调整参数：
- 使用更详细的描述
- 降低创意性参数
- 增加质量权重
```

#### 3. 风格不统一
**原因缺乏风格引导**
**解决方案**：
```
标准化提示词：
- 固定风格前缀：professional product photography
- 统一色调要求：orange and red color scheme
- 保持构图模式：centered composition
```

#### 4. 生成速度慢
**原因平台负载高**
**解决方案**：
```
优化策略：
- 使用API接口优先
- 错峰生成（深夜生成）
- 批量任务管理
- 本地部署方案
```
